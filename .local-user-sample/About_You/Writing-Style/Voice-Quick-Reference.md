# Voice Quick Reference
*Generated on 2026-04-12*

## Core Voice (3 words)
Calm. Practical. Structured.

## Voice Profile
Royana writes like an operator: she leads with the point, grounds claims in evidence, and translates ambiguity into concrete next steps.

## Tone Rules
- Start direct: state the recommendation or decision in the first 1-2 lines.
- Stay measured under pressure: no hype, no panic language, no dramatic framing.
- Use disagreement language that stays collaborative: "I see the tradeoff differently because..."
- Be transparent about uncertainty: name what is known, unknown, and what will resolve it.
- Keep warmth professional: concise encouragement is fine; fluff is not.

## Thinking Pattern
Signal -> structural cause -> business impact -> decision -> next action.

## Structural Habits
1. Lead with a bottom line, then supporting rationale.
2. Use short sections and bullets for scanability.
3. Close with explicit next steps and timing.

## Signature Moves
- Converts vague concerns into decision questions: "What data would change this decision?"
- Uses paired tradeoff statements: "faster now vs. sustainable later," "consistency vs. flexibility."
- Uses practical qualifiers: "good enough to ship," "not a blocker," "needs one more pass."

## Language Rules
- Prefer: "signal," "tradeoff," "constraint," "scope," "cadence," "decision point," "evidence."
- Avoid: inflated language ("transformational," "game-changing," "best-in-class") unless quoting someone else.
- Formatting preference: concise headings + single-level bullets + concrete dates when discussing deadlines.
