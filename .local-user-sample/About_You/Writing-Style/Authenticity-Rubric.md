# Writing Authenticity Rubric
*Generated on 2026-04-12*

## How to Use

Score each dimension 1-5. If any dimension is below 3, revise before sending.
Target: all dimensions 4+.

## Scoring Dimensions

### 1. Clarity and Directness
Does the text state the point early and avoid unnecessary detours?

| Score | Description |
|---|---|
| 1 | Buried point, vague ask, hard to follow |
| 3 | Main idea is present but delayed or diluted |
| 5 | Clear recommendation/ask appears immediately |

### 2. Practicality
Does the text translate ideas into actions, decisions, or timelines?

| Score | Description |
|---|---|
| 1 | Abstract commentary only |
| 3 | Some action language but not specific |
| 5 | Concrete actions, owners, and timing are explicit |

### 3. Evidence Discipline
Are claims anchored to facts, examples, or observed signals?

| Score | Description |
|---|---|
| 1 | Assertions without support |
| 3 | Mixed: some grounded points, some assumptions |
| 5 | Claims are consistently evidence-backed and transparent about unknowns |

### 4. Tone Under Pressure
Does the text stay calm, collaborative, and non-defensive?

| Score | Description |
|---|---|
| 1 | Reactive, sharp, or overconfident |
| 3 | Neutral but stiff or impersonal |
| 5 | Calm, respectful, and firm without escalation |

### 5. Structure and Scanability
Is the structure easy to skim for busy readers?

| Score | Description |
|---|---|
| 1 | Dense blocks, no hierarchy |
| 3 | Some structure but uneven flow |
| 5 | Strong headings, concise bullets, clear sequencing |

### 6. Tradeoff Awareness
Does the text acknowledge constraints and alternatives?

| Score | Description |
|---|---|
| 1 | One-sided argument, no constraints |
| 3 | Mentions tradeoffs but weakly |
| 5 | Explicitly frames options, risks, and decision rationale |

## Composite Score
- 24-30: Ready to send
- 18-23: Needs light editing
- Below 18: Rewrite
